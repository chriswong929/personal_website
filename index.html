Chrome Reminders

C3! Hackathon - February 2014

Chrome Reminders was my first experience with HTML, CSS, and JavaScript and was built at my first ever hackathon. This Chrome Extension allows users to set reminders within Chrome by implementing URL links that bring users back to a specific webpage. With Chrome Reminders, users can organize and manage internet tasks from right within the browser (no need to open external applications). In addition, your reminders are synced with your Google account and you can access them on any computer that has the extension installed. This app was selected as a finalist to present in front of judges at the end of the hackathon.

You can check out the Chrome Reminder Submission for the C3! Hackathon and see more pictures of the extension in action here.

Digital System Design

November 2013

As part of a school project, I helped design a large digital system in Verilog using programmable logic on an FPGA chip. The system utilized Altera's DE2 Board and accepted PS2 keyboard inputs and the output was displayed on the VGA. The project was a game inspired by Angry Birds and the core gameplay involved launching the iconic red bird and hitting moving targets. The digital system was based on state machine transitions and I personally wrote code to build the physics engine and the control system that governed how different components of the system interacted.
